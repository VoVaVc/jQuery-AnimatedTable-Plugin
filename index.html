Initating the plugin

To install the plugin you need two files.

  1. jQuery.js - the library itself, you can download it from here
  2. animatedTable.jquery.js - main plugin script

example:

<script src="jquery.js" type="text/javascript"></script>
<script src="animatedTable.jquery.js" type="text/javascript"></script>

insert this to your <head> and locate files in same directory or specify path to them

Optionaly you can include animatedTable.css stylesheet, it contains default style for your tables

Before using the plugin you need to initate it like this:

$('table').animatedTable()


 $('table') must be a table

to apply some options to plugin you need the following construction

$('table').animatedTable({
   debug: true,
   option1: value,
   option2: value
})


 list options like this and you are ready to go further

Plugin options

  • Debug
    • Type: boolean Default: false

    If set to true errors will be logged

  • Duration
    • Type: numeric Default: 500

    Sets up animation speed

  • wrapperClass
    • Type: string Default: animatedTable

    Class, that will be added to wrapper divs, which control animation

  • tableClass
    • Type: string Default: animatedTableMain

    Class, that will be added to table

  • cellAnimation
    • Type: custom string Default: horizontal

    Possible values:

    1. horizontal
    2. vertical

    Defines animation for add/remove cell method, if set to vertical cell will appear with vertical animation, else it will run horizontally

  • dragCell
    • Type: boolean Default: true

    If set to true your table cells can be dragged around

  • dragCellDelay
    • Type: numeric Default: 150

    Specifies delay between mousedown and drag start, works only if dragCell set to true

  • dragCellSnap
    • Type: boolean Default: false

    If set to true, after delay gone cell will be snapped to cursor even after mouseup, to release drag you need to click second time, works only if dragCell set to true

  • dragCellCopy
    • Type: boolean Default: false

    If set to true, drag function won't exchange cells, but copy dragging one to release point, works only if dragCell set to true

  • dragZindex
    • Type: numeric Default: 999

    Specifies z-index for dragging cell, works only if dragCell set to true

  • exchangeFade
    • Type: boolean Default: true

    If set to true exchanging cells will be animated with fade

  • exchangeFadeDuration
    • Type: numeric Default: 500

    Specifies animation speed for exchanging cells

Plugin methods

  • init
    • $(element).animatedTable(options)

    Arguments:

    1. element - can be selector, node or jquery object
    2. options - object, contains different options, full list of avalible options you can watch here

    Initates plugin, on this step each cell content wraps in a special div, that makes animation possible. Run this constructor before calling any other method.

    example:

    $('table').animatedTable({
       debug: true,
       dragCell: true
       duration: 1000,
    })

     

  • addRow
    • $(element).animatedTable('addRow', data, trIndex)

    Arguments:

    1. element - can be selector, node or jquery object
    2. data - selector, node/nodes or jquery object
    3. trIndex - index number, specifying after which row new row must be appended

    Adds row with specified params and smooth animation

    example 1, this will clone first row to bottom:

    $('table').animatedTable('addRow', $('table tr:nth-child(1)'), 999)

    example 2, this will create a new row and place it after second row:

    $('table').animatedTable('addRow', '<td>some</td><td>text</td><td>here</td>', 1)

     

  • removeRow
    • $(element).animatedTable('removeRow', identifier)

    Arguments:

    1. element - can be selector, node or jquery object
    2. identifier - selector, existing node, jquery object or index number

    Removes row by identifier with smooth animation

    example 1, this will remove first row:

    $('table').animatedTable('removeRow', $('table tr:nth-child(1)'))

    example 2, this will remove second row:

    $('table').animatedTable('removeRow', 2)

     

  • addCol
    • $(element).animatedTable('addCol', colHead, colIndex, data, isBefore)

    Arguments:

    1. element - can be selector, node or jquery object
    2. colHead - selector, node, jquery object or text, that will be appeared as column head
    3. colIndex - index number, after which will be added a new column
    4. data - selector, nodes, jquery objects or array of td elements, which will be added in each row
    5. isBefore - optional, if set to true columnt will appear before specified index instead of after

    Adds column with specified params and smooth animation

    example 1, this will add a column to the first place

    $('table').animatedTable('addCol', 'new Column', 0, ['<td>123</td>', '124', '235', '226'], true)

    example 2, this will add a new column as third one:

    $('section.show table').animatedTable('addCol', 'new Column', 1, $('<td>1</td><td>2</td><td>3</td><td>4</td>'))

     

  • removeCol
    • $(element).animatedTable('removeCol', index)

    Arguments:

    1. element - can be selector, node or jquery object
    2. index - column index number

    Removes column with smooth animation

    example, this will remove first column

    $('table').animatedTable('removeCol', 0)

     

  • addCell
    • $(element).animatedTable('addCell', trIndex, tdIndex, cell, isBefore)

    Arguments:

    1. element - can be selector, node or jquery object
    2. trIndex - row index number, which will receive a new cell
    3. tdIndex - cell index number, after which will be added a new cell
    4. cell - selector, node, jquery object or text, this is a content, which will be added to new cell
    5. isBefore - optional, if set to true new cell will be appended before specified tdIndex

    Adds cell with smooth animation

    example 1, this will add new cell to the first place in first row

    $('table').animatedTable('addCell', 0, 0, '<span>I am new here</span>', true)

    example 2, this will add new cell to the third place in second row

    $('table').animatedTable('addCell', 1, 1, 'Hi, me too')

     

  • removeCell
    • $(element).animatedTable('removeCell', trIndex, tdIndex)

    Arguments:

    1. element - can be selector, node or jquery object
    2. trIndex - can be selector, exsisting node, jquery object or index number of row
    3. tdIndex - index number of cell to remove, required only if trIndex is index number

    Removes cell with smooth animation. Warning, you can pass this method with 1 or 2 arguments, if argument is single, you need to specify a selector for your td to remove, else method accepts index of tr and td

    example 1, this will remove first cell in first row

    $('table').animatedTable('removeCell', 0, 0)

    example 2, this will remove row with id='removeMe'

    $('table').animatedTable('removeCell', $('#removeMe'))

     

  • exchangeCells
    • $(element).animatedTable('exchangeCells', cell1, cell2, clone, preventFade)

    Arguments:

    1. element - can be selector, node or jquery object
    2. cell1 - selector, existing node or jquery object, this cell will be placed to cell2 position or copied there
    3. cell2 - selector, existing node or jquery object, this cell will be placed to cell1 position or overwrited
    4. clone - if set to true cell1 will be copied and overwrite cell2
    5. preventFade - column index number

    Exchange or clone colums with or without animation

    example 1, this will exchange first cell in first row and last cell in first row

    $('table').animatedTable('exchangeCells', $('table tr:nth-child(1) td:nth-child(1)'), $('table tr:nth-child(1) td:last'))

    example 2, this will clone first cell in first row to last cell in last row without animation

    $('table').animatedTable('exchangeCells', $('table tr:nth-child(1) td:nth-child(1)'), $('table tr:last td:last'), true, true)

     

  • option
    • $(element).animatedTable('option', name, value)

    Arguments:

    1. element - can be selector, node or jquery object
    2. name - option name string or array of options
    3. value - option value, required only if name is a string

    Applies option to specified table. Full list of options avalible here

    example 1, this will apply dragCell mode

    $('table').animatedTable('option', 'dragCell', true)

    example 2, this will apply multiple options

    $('table').animatedTable('option', {
       'dragCell': true,
       'dragCellCopy': true,
       'dragCellSnap': true,
       'dragCellDelay': 1000
    })

     

  • destroy
    • $(element).animatedTable('destroy')
    1. element - can be selector, node or jquery object

    Simply destroys plugin
